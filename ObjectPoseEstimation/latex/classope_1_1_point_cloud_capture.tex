\subsection{Detailed Description}
Captures X\-Y\-Z\-R\-G\-B point clouds from the Kinect. 

\begin{DoxyNote}{Note}
The coordinate system for the captured point clouds is as follows\-: x-\/axis -\/$>$ right, y-\/axis -\/$>$ down, z-\/axis -\/$>$ points into scene.
\end{DoxyNote}
